This library provides convenience APIs to access the MDS service. It allows for querying the MDS and adding, modifying or deleting MDS entries. The library is based on the JNDI library and enables communication with the MDS over a SASL interface (secure MDS) from both JNDI and the Netscape Directory SDK.
